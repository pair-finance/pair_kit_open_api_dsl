# PairKit::OpenApi::Dsl

Ruby DSL to build Open Api Specifications

---
**NOTE**

This gem is under heavy developmetn. Don't use it now, wait until v.0.1.0

---


## Usage

In your ```Gemfile```:

```ruby
gem 'pair_kit_open_api_dsl'
```

In your code:

```ruby
require 'pari_kit/open_api/dsl'
```

### Example

```ruby
dsl = PairKit::OpenApi::Dsl.new


```
